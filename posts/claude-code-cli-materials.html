Below are free, copy/paste-friendly materials you can reuse while learning Claude Code CLI. They’re designed for low-bandwidth learning and small study groups.

Prompt templates (copy/paste)

Template A: understand a codebase quickly

Context: I’m new to this project.
Task: explain how the app starts and where routing happens.
Output:
- 5 bullet summary
- key files to read (with paths)
- “try this next” checklist (3 items)

Template B: make a safe change

Goal: <what you want>
Constraints:
- keep changes minimal
- don’t add new dependencies unless necessary
Definition of done:
- <how to verify>
Ask: propose a plan, then implement.

Template C: debugging

Bug: <what happens>
Expected: <what should happen>
Evidence:
- exact error message
- steps to reproduce
Ask:
- list likely causes (top 3)
- pick 1 and propose a fix

Filled example (ready to reuse)

Goal: Add a search box to filter posts by title and tags.
Constraints:
- keep changes minimal
- no new dependencies
Definition of done:
- typing in the search box filters the visible post cards instantly
- empty search shows all posts again
Ask: propose a plan, then implement.

Filled debug example (ready to reuse)

Bug: Clicking a post shows “Post not found”.
Expected: The post page loads the content.
Evidence:
- URL: /post.html?post=welcome
- Console shows: “Post not found”
Ask:
- list likely causes (top 3)
- pick 1 and propose a fix

Review checklist (before you accept changes)

  • Scope: did it change only what you asked for?
  • Safety: no secrets added (API keys, passwords, tokens)?
  • Clarity: do you understand the main changes?
  • Run it: does it work after the change?
  • Shareable: can another student read it and learn from it?

Practice tasks (offline-friendly)

  1. Documentation exercise: ask Claude Code to write a short README for a folder, then verify every step works.
  2. Small refactor: rename a function across files and ensure nothing breaks.
  3. Bug hunt: intentionally add a tiny bug, then use the debugging template to find it.
  4. UI polish: improve spacing/contrast on a page while keeping it accessible.

Related learning materials (free)

Pair Claude Code practice with basic foundations:

  • Git basics (commits, diffs, branches)
  • Command line basics (folders, files, running scripts)
  • HTML/CSS/JS fundamentals (for web projects)
  • Python fundamentals (for scripting)

Tip: keep a “study pack” folder you can zip and share offline with your local group.
